2. **Estandarización de Datos Generales:** Todas las plantillas (desde la simplificada hasta la de auditoría) muestran ahora la información básica del producto (Nombre, Código, Cantidad, UM) de forma consistente.

4. **Resiliencia de Diseño:** Implementación de `maxWidth` para evitar que títulos largos rompan el layout.
