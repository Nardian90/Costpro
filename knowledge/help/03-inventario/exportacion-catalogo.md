## 1. Introducción al Sistema de Exportación

CostPro incluye un **motor de exportación de catálogos integrado** que permite generar materiales profesionales de marketing directamente desde los datos de tu inventario. En lugar de diseñar catálogos desde cero en herramientas externas, el sistema toma los productos, precios, imágenes y datos de tu tienda para producir archivos listos para distribución.

El exportador soporta **4 plantillas** diseñadas para diferentes canales de comunicación y audiencias:

- **WhatsApp**: Grilla visual rápida para compartir por mensajería.
- **Instagram**: Diapositivas individuales optimizadas para formato de carrusel.
- **Lista de Precios**: Tabla densa profesional en formato PDF para clientes mayoristas o B2B.
- **Catálogo Elegante**: Publicación tipo revista con portada profesional, grilla de productos y pie de página corporativo.

Cada plantilla genera un archivo de alta calidad que puedes descargar y distribuir inmediatamente sin necesidad de edición posterior.

---

## 2. Acceder al Exportador de Catálogos

El exportador se accede desde el módulo de ventas de la siguiente manera:

```
MULTI-TIENDA > Ventas > Catálogo de Ventas
```

Una vez en la vista del Catálogo de Ventas, localiza el botón **"Exportar Catálogo"** en la barra de herramientas superior. Al hacer clic, se abrirá el **Modal de Exportación** que contiene todas las opciones de plantilla, personalización y filtros.

> **Tip**: También puedes acceder al exportador desde la tabla de ventas usando la acción contextual "Exportar" disponible en la barra superior de la vista de catálogo.

## 3. Plantillas Disponibles

### 3.1 Catálogo WhatsApp (JPG)

**Formato de salida:** Imagen JPG (una sola página)
**Ideal para:** Grupos de WhatsApp, estados, mensajes directos

Esta plantilla genera una **grilla visual de 12 productos** organizados en una cuadrícula compacta (3 columnas x 4 filas). Cada tarjeta muestra:

- Imagen del producto (centrada y recortada)
- Nombre del producto (truncado si es muy largo)
- Precio formateado con símbolo de moneda

El fondo utiliza el **color primario seleccionado** y las tarjetas tienen esquinas redondeadas con fondo blanco. El diseño está optimizado para verse bien en pantallas de móviles y se comparte fácilmente como imagen en chats de WhatsApp.

**Características:**
- Genera una **única imagen JPG** con todos los productos.
- Los productos sin imagen muestran un marcador de posición con la primera letra del nombre.
- El número de productos shown es exactamente 12 por página. Si tienes más productos, se generarán múltiples imágenes (una por cada grupo de 12).
- El nombre de archivo incluye la fecha de generación.

### 3.2 Carrusel Instagram (JPG)

**Formato de salida:** Múltiples imágenes JPG (una por producto)
**Ideal para:** Publicaciones de carrusel en Instagram, Facebook, Stories

Esta plantilla genera **una diapositiva individual por cada producto** seleccionado, optimizada para la proporción vertical de Instagram (4:5). Cada diapositiva incluye:

- Imagen del producto a pantalla completa como fondo
- Nombre del producto en la parte inferior con overlay de color
- Precio formateado en texto grande y legible
- Barra inferior con el color de marca seleccionado

**Características:**
- Genera **N archivos JPG** donde N es el número de productos seleccionados.
- La primera diapositiva es la **portada del carrusel** con el nombre de la tienda y el número de productos.
- El formato es óptimo para subir directamente a Instagram sin recortes.
- Cada imagen incluye sutil degradado en la parte inferior para asegurar la legibilidad del texto sobre cualquier imagen de producto.

### 3.3 Lista de Precios (PDF)

**Formato de salida:** Documento PDF
**Ideal para:** Clientes mayoristas (B2B), distribución por correo electrónico, archivo interno

Esta plantilla genera un **documento PDF profesional** con formato de tabla densa. Es la opción preferida cuando necesitas comunicar precios de forma clara y estructurada a clientes comerciales o para archivo interno.

**Características:**
- **Hasta 20 productos por página** en formato de tabla.
- Columnas: código, nombre del producto, categoría, precio de costo, precio de venta y margen.
- Encabezado con logo y nombre de la tienda.
- Tabla con bordes limpios y alternancia de colores para facilitar la lectura.
- Si el catálogo tiene más de 20 productos, se generan múltiples páginas con encabezado repetido.
- Formato A4 optimizado para impresión.

### 3.4 Catálogo Elegante (PDF)

**Formato de salida:** Documento PDF premium
**Ideal para:** Presentaciones a clientes, material de ventas impreso, portfolios

Esta es la plantilla **más completa y profesional**. Genera una publicación tipo revista con:

- **Portada profesional** con banda de marca, producto destacado, avatar promocional y footer corporativo (ver sección 4.4 para detalles).
- **Grilla de productos** (4 por fila, 16 por página) con tarjetas que incluyen imagen, nombre y precio.
- **Pie de página** en cada página con datos de contacto de la tienda.
- Diseño de alta calidad con elementos decorativos y composición equilibrada.

**Características:**
- Incluye automáticamente una **portada completa** como primera página.
- Los productos se muestran en tarjetas con sombra sutil y esquinas redondeadas.
- El fondo de la portada utiliza el color de marca con degradado.
- El avatar promocional se muestra como banner en la portada (ver sección 4.2).
- Un producto se destaca como "producto hero" en la portada.
- Soporte para múltiples páginas cuando hay más de 16 productos.

## 4. Personalización Visual

### 4.1 Paleta de Colores

El exportador incluye **8 colores predefinidos** que aplican a la banda de marca, fondos, acentos y elementos decorativos de todas las plantillas:

| Color | Muestra | Uso Recomendado |
|-------|---------|------------------|
| **Verde** | Tono natural y fresco | Productos orgánicos, alimentación |
| **Azul** | Profesional y confiable | Servicios, tecnología, corporativo |
| **Rojo** | Energiante y llamativo | Ofertas, promociones, retail |
| **Naranja** | Cálido y dinámico | Restaurantes, entretenimiento |
| **Morado** | Elegante y sofisticado | Moda, lujo, boutique |
| **Teal** | Moderno y equilibrado | Salud, bienestar, diseño |
| **Grafito** | Sobrio y minimalista | Industrial, construcción, B2B |
| **Dorado** | Premium y exclusivo | Joyería, alta gama, eventos |

Para seleccionar un color, haz clic en el círculo de color correspondiente dentro del modal de exportación. El cambio se refleja **inmediatamente** en la vista previa de la plantilla.

> **Tip**: Elige un color que coincida con la identidad visual de tu marca. Si tu logo es verde, selecciona el color Verde para mantener consistencia en todos los materiales generados.

### 4.2 Avatar Promocional

El avatar es una **imagen de banner promocional** que se incluye en la portada del Catálogo Elegante. CostPro incluye **6 avatares predefinidos**:

| Avatar | Descripción |
|--------|-------------|
| **Ven tenemos lo mejor** | Banner general de invitación a comprar |
| **Ofertas especiales** | Banner para promociones y descuentos |
| **Nuevas llegadas** | Banner para productos recién ingresados |
| **Liquidación total** | Banner para ventas de liquidación |
| **Lo mejor del mes** | Banner destacando productos del mes |
| **Envío gratis** | Banner promocionando envío sin costo |

El avatar se muestra como una franja horizontal en la sección inferior de la portada del Catálogo Elegante, entre el producto destacado y la fila de productos de vista previa. Para seleccionarlo, haz clic en la miniatura del avatar dentro del modal de exportación.

> **Nota**: Los avatares solo están disponibles para la plantilla **Catálogo Elegante**. Las demás plantillas no incluyen avatar en su diseño.

### 4.3 Datos de Marca

Independientemente de la plantilla seleccionada, el exportador utiliza los **datos de marca** configurados en tu tienda:

| Campo | Se usa en |
|-------|-----------|
| **Nombre de la tienda** | Encabezado de todas las plantillas, portada |
| **Teléfono** | Pie de página del Catálogo Elegante, Lista de Precios |
| **Correo electrónico** | Pie de página del Catálogo Elegante |
| **Dirección** | Pie de página del Catálogo Elegante |
| **WhatsApp** | Pie de página del Catálogo Elegante |
| **Sitio web** | Pie de página del Catálogo Elegante |
| **Logo** | Encabezado de Lista de Precios, marca de agua en Catálogo Elegante |

Estos datos se extraen automáticamente de la configuración de la tienda activa. Para modificarlos, navega a **CONFIGURACIÓN > Administrativa > Tiendas** y edita la tienda correspondiente.

### 4.4 Portada Profesional (Catálogo Elegante)

La portada del Catálogo Elegante es el elemento más elaborado del sistema de exportación. Se compone de las siguientes secciones de arriba hacia abajo:

**1. Banda superior de marca:** Franja horizontal con el color seleccionado que contiene el nombre de la tienda en texto blanco de gran tamaño, centrado y en negrita.

**2. Círculos decorativos:** Elementos geométricos semi-transparentes que añaden profundidad visual a la esquina superior de la portada.

**3. Logo de tienda (inicial):** Se muestra la primera letra del nombre de la tienda dentro de un círculo blanco, posicionado como elemento visual de marca.

**4. Producto hero:** Un producto del catálogo se presenta en grande como imagen destacada, con sombra y marco redondeado. El sistema selecciona automáticamente el primer producto con imagen disponible.

**5. Banner de avatar:** El avatar promocional seleccionado se muestra como franja horizontal debajo del producto hero.

**6. Fila de productos de vista previa:** Una hilera de hasta 4 productos en miniatura como adelanto del contenido del catálogo.

**7. Pie de página (3 filas):**
- **Fila 1:** Dirección de la tienda
- **Fila 2:** Teléfono y correo electrónico
- **Fila 3:** WhatsApp y sitio web

> **Tip**: Para obtener la mejor portada posible, asegúrate de que tus productos tengan **imágenes de alta calidad** asignadas. El producto hero se selecciona automáticamente, pero una buena imagen marca la diferencia entre una portada amateur y una profesional.

## 5. Filtros y Selección de Productos

Antes de exportar, puedes **filtrar los productos** que se incluirán en el catálogo:

| Filtro | Descripción |
|--------|-------------|
| **Por categoría** | Selecciona una o varias categorías para incluir solo los productos de esas categorías. |
| **Por nombre** | Escribe parte del nombre para filtrar productos que coincidan con la búsqueda. |
| **Por stock** | Filtra productos con stock disponible, sin stock o todos. |
| **Todos** | Incluye todos los productos del catálogo sin filtros. |

Los filtros se aplican dentro del Catálogo de Ventas antes de abrir el modal de exportación. Solo los productos visibles en la tabla de catálogo al momento de exportar serán incluidos en el archivo generado.

> **Importante**: Verifica que los productos seleccionados tengan **precios asignados e imágenes** antes de exportar. Los productos sin precio mostrarán "$0.00" y los sin imagen mostrarán un marcador de posición genérico, lo cual afecta la calidad visual del catálogo final.

## 6. Flujo de Exportación Paso a Paso

Sigue estos pasos para generar un catálogo desde cero:

### Paso 1: Preparar los Productos

1. Asegúrate de que los productos tengan **nombre, precio e imagen** completos.
2. Verifica que las **categorías** estén asignadas correctamente (para filtrado).
3. Confirma que los **datos de marca** de la tienda estén actualizados.

### Paso 2: Acceder al Catálogo de Ventas

1. Navega a **MULTI-TIENDA > Ventas > Catálogo de Ventas**.
2. Revisa la tabla de productos para confirmar que se muestran correctamente.

### Paso 3: Aplicar Filtros (Opcional)

1. Usa los controles de filtro en la barra superior del catálogo.
2. Selecciona la categoría, texto de búsqueda o estado de stock deseado.
3. Verifica que los productos filtrados son los que deseas exportar.

### Paso 4: Abrir el Modal de Exportación

1. Haz clic en el botón **"Exportar Catálogo"**.
2. Se abrirá el modal con las opciones de plantilla.

### Paso 5: Seleccionar Plantilla

1. Elige una de las 4 plantillas disponibles:
   - WhatsApp (JPG) para compartir por chat
   - Carrusel Instagram (JPG) para redes sociales
   - Lista de Precios (PDF) para clientes B2B
   - Catálogo Elegante (PDF) para material premium
2. Observa la miniatura de vista previa que se actualiza con tu selección.

### Paso 6: Personalizar

1. Selecciona el **color de marca** haciendo clic en uno de los 8 círculos de color.
2. Si elegiste Catálogo Elegante, selecciona un **avatar promocional** de las 6 opciones.
3. Verifica que los datos de marca (nombre, teléfono, email) sean correctos.

### Paso 7: Generar y Descargar

1. Haz clic en el botón **"Exportar"**.
2. Espera a que el sistema genere el archivo (el tiempo depende de la cantidad de productos y la plantilla).
3. El archivo se descargará automáticamente a tu equipo.
4. Distribúyelo por el canal correspondiente.

## 8. Buenas Prácticas

> **📸 Calidad de imágenes:** Usa imágenes de al menos 800x800 píxeles para productos. Las imágenes de baja resolución se pixelan al ampliarlas en el catálogo, especialmente en el Catálogo Elegante donde el producto hero ocupa una gran porción de la portada.
>
> **🏷️ Categorización consistente:** Mantén una estructura de categorías clara y consistente. Los filtros por categoría son la forma más rápida de seleccionar productos para un catálogo temático (ej. "Verano 2024", "Nuevos Ingresos", "Liquidación").
>
> **💰 Precios actualizados:** Exporta el catálogo inmediatamente después de actualizar precios. Los precios se toman del momento exacto de la exportación, no se actualizan automáticamente en archivos ya generados.
>
> **🎨 Identidad de marca consistente:** Usa siempre el mismo color de marca para todas las exportaciones. Esto refuerza el reconocimiento de tu marca entre los clientes que reciben catálogos frecuentes.
>
> **📊 Segmentación por audiencia:** No exportes todo el catálogo para todos los canales. Usa filtros para crear catálogos segmentados: productos en oferta para WhatsApp, novedades para Instagram, listas completas para B2B.
>
> **⏰ Frecuencia de actualización:** Establece una periodicidad para actualizar los catálogos. Un catálogo con precios desactualizados genera frustración en los clientes y puede causar problemas de servicio al cliente. Como regla general, actualiza al menos una vez por semana o cuando haya cambios significativos de precio o stock.

## 9. Solución de Problemas

| Problema | Causa probable | Solución |
|----------|----------------|-----------|
| El catálogo muestra "$0.00" en algunos productos | El producto no tiene precio de venta asignado | Edita el producto en el Catálogo Maestro y asigna un precio de venta |
| Las imágenes aparecen borrosas | La resolución de la imagen original es baja | Sube imágenes de mayor resolución (mínimo 800x800 px) |
| El avatar no aparece en la portada | Se seleccionó una plantilla diferente a Catálogo Elegante | El avatar solo está disponible para la plantilla Catálogo Elegante (PDF) |
| El archivo es muy pesado | Demasiados productos con imágenes de alta resolución | Reduce la cantidad de productos por exportación o usa la Lista de Precios (menos imágenes) |
| Los colores no coinciden con mi marca | No se seleccionó el color correcto | Haz clic en el círculo de color que mejor represente tu marca dentro del modal de exportación |
| El nombre de la tienda aparece incorrecto | Los datos de marca no están configurados | Navega a CONFIGURACIÓN > Administrativa > Tiendas y edita los datos de la tienda activa |
